About
--------

A Plone `Snap Package <http://snapcraft.io/>`_.

Currently the main focus is to learn more about `snaps <http://snapcraft.io/>`_ and to figure out if that would be something for `Plone <https://plone.org>`_ in order to make installations 'easier' and 'faster'.
